Point Blank Server Emulator

Download this project as a .zip file Download this project as a tar.gz file

What's it this?

This is server emulator MMO-FPS Point Blank.

Functional

  1. PvP 8 x 8
  2. PvE 4 x BOTS
  3. Shop (~85%)
  4. Clans (~45-50%)
  5. Inventory (~80%)
  6. Friends (~60%)
  7. Private Messages (~50%)
  8. Missions (~80%) Other stuff

How to setup and play

  1. Install MySQL
  2. Install .NET Framework
  3. Change settings from Settings.ini to your params
  4. Install Apache
  5. Install site from /Website/
  6. Change configs for site (core/config.php)
  7. Create account (localhost/reg.php)
  8. Run servers (start_all_servers.cmd)
  9. If you correctly set up, all will work
